# ASD.NES
### Emulator of the "Nintendo Entertainment System" game console.

#### Capabilities:
- Almost all code is a cross-platform library (.NET Standard 1.1).
- View (WPF): Windows 10 or 7 with .NET Framework 4.5.
- Sound: all APU channels supported (pulse, triangle, noise, DMC).
- Two players; key mapping configurable via **SETTINGS** — saved to `ASD.NES.Settings.xml` next to the executable.
